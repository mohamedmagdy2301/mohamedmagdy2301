## 👨‍💻 About Me

I'm a **Full-Stack & Mobile Developer** from **Shibin El Kom, Egypt** 🇪🇬, passionate about building cross-platform products that are fast, clean, and user-first.

- 🚀 Founder & Developer of **[Escore](https://escore.app)** — a live esports platform (Flutter + React + Flask)
- 📱 Building cross-platform mobile apps with **Flutter & Dart**
- 🌐 Crafting web dashboards & frontends with **React**
- ⚙️ Engineering backends with **Python Flask** and **Node.js**
- 🤖 Integrating AI features using **Gemini 2.0 Flash** and **Whisper AI**
- 🌍 Hosting production apps on **DigitalOcean**
- 📐 Obsessed with clean architecture, RTL Arabic-first UX, and developer tooling
